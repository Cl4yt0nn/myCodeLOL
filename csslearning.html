HTML5 is a markup language used for structuring and presenting content on the World Wide Web. It was first released on 22 January 2008. Its goals were to improve the language with support for the latest multimedia and other new features; to keep the language both easily readable by humans and consistently understood by computers and devices such as web browsers. HTML5 is used in the vast majority of websites on the web, and CSS is used in hand with HTML5 to decorate these websites and improve the overall User Experience. CSS contributed to HTML by making tags/commands dedicated to customizing the look of the website, such as <section>, <article>, <header>, and many more. It also implemented many new commands, some being the <video> and <audio> tags, the <link> tag, and the <canvas> tag. HTML is necessary to create web pages as that's its real main purpose. Even website-making websites use HTML to create said sites and the site itself uses HTML to run. JavaScript, PHP, and other coding languages are actually known as scripting languages, which are used with HTML but are more expansive than it.
